## 获取课程名称和安排表的接口

> 维护人员：裴雷
>
> 创建时间：2020-11-03

### 接口简介

查询数据库中课程名称和课程安排表

### 接口详情

#### 请求地址

/get_course_plan/:id

#### 请求类型

GET

#### 请求参数

| 参数名 | 类型   | 必填 | 描述         |
| ------ | ------ | ---- | ------------ |
| id     | string | 是   | 课程的唯一id |

#### 返回正确JSON示例

```node.js
{
	"status": {
		"code": 200,
		"msg": "ok"
	},
	"data": {
		"id": "6",             //课程id
		"name": "腿部康复",        //课程名称
		"length":"20",        //课程的总进度
		"actions":[
            {
                "id":"6"       //动作的id
            }
    	],
		"logo":"图片url",           //课程logo的url
		"info":"一段介绍"          //课程介绍
	}
}
```

#### 返回错误JSON示例

```node.js
{
	"status": {
		"code": 500,
		"msg": "error"
	}
}
```
